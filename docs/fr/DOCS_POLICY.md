# 🔒 Politique de langue de la documentation

- 🇫🇷 Le français est la **source de vérité**
- 🇬🇧 L’anglais est une **traduction maintenue**

- Toute modification conceptuelle doit être validée en français en premier
- L’anglais doit être mis à jour en conséquence

- Les traductions dans d’autres langues sont les bienvenues, mais doivent être basées sur la version française
- En cas d’incohérence, la version française fait toujours foi
